# Project Templates & Blueprints

> **Reference Guide**

This document tracks the available project templates (Blueprints) available in this framework.
The actual source files for these templates are located in **`.agent/blueprints/`**.

## Available Blueprints

### 📂 01-Web and Apps

- **Full Stack App**: Next.js + Supabase + Tailwind.
- **Static Site**: Astro / Hugo.
- **Mobile App**: React Native / Expo.

### 📂 02-Games

- **Unity**: C# Scripts and Project Structure.
- **Godot**: GDScript Templates.

### 📂 03-Trading and Finance

- **Trading Bot**: Python/Pandas Algo Templates.
- **DeFi Dashboard**: Web3.js + React.

### 📂 05-AI and ML

- **RAG Pipeline**: LangChain/LlamaIndex setup.
- **Agent Swarm**: Multi-agent orchestration template.

---

## How to Use

1. Navigate to `.agent/blueprints/[Category]/[Blueprint]/`.
2. Copy the `starter-context.md` content.
3. Paste it into your project's `.agent/docs/0-context/project-context.md`.
