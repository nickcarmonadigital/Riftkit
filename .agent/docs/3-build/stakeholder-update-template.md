## RACI Matrix

> **What is RACI?** Clarifies who does what on each task.
> - **R**esponsible: Does the work
> - **A**ccountable: Owns the outcome (only one per task)
> - **C**onsulted: Gives input before the work
> - **I**nformed: Told after the work is done
