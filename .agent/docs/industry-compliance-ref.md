# Industry Compliance Quick Reference

**Purpose**: Quick reference for industry-specific compliance requirements.

---

## 🔍 How to Use

1. Find your client's industry below
2. Review key compliance concerns
3. Incorporate requirements into build

---

## Healthcare / Medical

| Requirement | Description |
|-------------|-------------|
| **HIPAA** | Protected Health Information (PHI) handling |
| **BAA Required** | Business Associate Agreement if handling PHI |
| **No PHI in Forms** | Avoid collecting medical info in standard forms |
| **Secure Hosting** | HIPAA-compliant hosting if storing data |

**Safe Approach**: Use booking systems without medical intake; direct patients to call for medical questions.

---

## Legal / Law Firms

| Requirement | Description |
|-------------|-------------|
| **Attorney Advertising Rules** | State bar requirements vary |
| **Disclaimers** | "This is not legal advice" messaging |
| **Confidentiality** | Secure contact forms |
| **State-Specific** | Check specific state bar association rules |

**Safe Approach**: Include proper disclaimers; consult client's bar association guidelines.

---

## Financial Services

| Requirement | Description |
|-------------|-------------|
| **SEC/FINRA** | Investment advisor regulations |
| **Disclaimers** | Required performance disclosures |
| **Data Protection** | PCI-DSS for payment handling |
| **No Guarantees** | Cannot promise returns |

**Safe Approach**: Use proper disclaimers; avoid performance claims; use compliant payment processors.

---

## Real Estate

| Requirement | Description |
|-------------|-------------|
| **Fair Housing Act** | No discriminatory content |
| **License Display** | Broker license numbers required |
| **Equal Opportunity** | Include Equal Housing logo |
| **Data Privacy** | Lead capture regulations |

**Safe Approach**: Display required licensing; include Fair Housing compliance statements.

---

## Food & Restaurant

| Requirement | Description |
|-------------|-------------|
| **Allergen Info** | FDA allergen disclosure |
| **Nutrition** | Calorie counts (if chain) |
| **ADA Compliance** | Accessible website and menus |
| **Health Permits** | Display licensing if required |

**Safe Approach**: Include allergen notices on menus; ensure PDF menus are accessible.

---

## E-Commerce / Retail

| Requirement | Description |
|-------------|-------------|
| **PCI-DSS** | Payment card handling |
| **Return Policy** | Clear, accessible policies |
| **Shipping Info** | Delivery estimates |
| **Privacy Policy** | Required for data collection |
| **Terms of Service** | User agreement |

**Safe Approach**: Use established payment processors (Stripe, PayPal); include required policy pages.

---

## Salons / Personal Services

| Requirement | Description |
|-------------|-------------|
| **State Licensing** | Display required licenses |
| **Service Disclaimers** | Allergy/sensitivity notices |
| **Booking Terms** | Cancellation policies |

**Safe Approach**: Include cancellation policies; note allergy considerations for chemical services.

---

## Education / Training

| Requirement | Description |
|-------------|-------------|
| **FERPA** | Student data protection (if applicable) |
| **Accessibility** | ADA/WCAG compliance |
| **Copyright** | Licensed content only |
| **Refund Policy** | Clear terms for courses |

**Safe Approach**: Use accessible design; include clear refund/cancellation policies.

---

## General Website Requirements

These apply to ALL industries:

| Requirement | Description |
|-------------|-------------|
| **Privacy Policy** | Required if collecting any data |
| **Terms of Service** | Required for user-generated content |
| **Cookie Consent** | Required in EU (GDPR) |
| **ADA/Accessibility** | WCAG 2.1 AA compliance |
| **SSL/HTTPS** | Required for any forms |
| **Contact Info** | Business address if required by state |

---

## When in Doubt

1. **Ask the client** about their specific licensing requirements
2. **Check state regulations** - they vary significantly
3. **Consult a lawyer** for complex compliance needs
4. **Use compliant third-party tools** (payment, booking, forms)

---

*Note: This is a quick reference, not legal advice. Always verify requirements with proper legal counsel.*
