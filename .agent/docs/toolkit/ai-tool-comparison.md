## Decision Tree

> When you are unsure which tool to use, follow this flowchart.

```
Start: What kind of task is it?
│
├─ Writing or planning?
│  ├─ Long document (>2000 words)? → Claude (long context)
│  ├─ Short email/message? → ChatGPT (fast)
│  └─ Needs real-time data? → Perplexity
│
├─ Coding?
│  ├─ Quick autocomplete? → GitHub Copilot
│  ├─ Multi-file changes? → Cursor / Claude Code
│  ├─ Need explanation? → Claude or ChatGPT
│  └─ Sensitive/private code? → Local LLM (Ollama)
│
├─ Research?
│  ├─ Need citations? → Perplexity
│  ├─ Deep analysis? → Claude
│  └─ Quick fact check? → ChatGPT with browsing
│
├─ Visual / design?
│  ├─ Generate images? → Midjourney / DALL-E
│  └─ UI mockups? → Figma AI / v0
│
└─ Data?
   ├─ Large dataset? → Gemini (large context)
   └─ Quick chart? → ChatGPT Code Interpreter
```
