# Long-Form Video Protocol

**Purpose**: System for creating high-quality long-form video content (1-6 hours).

---

## 🎯 When to Use

- Creating tutorial/build videos
- Recording educational content
- Making portfolio/demo videos

---

## Pre-Production Checklist

### 1. Topic Selection

| Criteria | Check |
|----------|-------|
| Topic has clear outcome | ☐ |
| Can be completed in one session | ☐ |
| Has visual/coding component | ☐ |
| Would be valuable to audience | ☐ |

### 2. Preparation

| Task | Done |
|------|------|
| Outline the build/tutorial | ☐ |
| Test all tools/dependencies | ☐ |
| Create starting template if needed | ☐ |
| Prepare reference materials | ☐ |
| Set up clean development environment | ☐ |

### 3. Technical Setup

| Task | Done |
|------|------|
| Screen recording software ready | ☐ |
| Microphone tested | ☐ |
| Notifications disabled | ☐ |
| Recording resolution set (1080p/4K) | ☐ |
| Storage space available | ☐ |

---

## During Recording

### Best Practices

1. **Talk through your thinking** - Explain decisions as you make them
2. **Don't hide mistakes** - Show debugging process
3. **Take breaks** - Stand up every 30-45 minutes
4. **Bookmark key moments** - Note timestamps for editing

### Pacing

| Type | Recommendation |
|------|----------------|
| Explanation | Slow down, be clear |
| Coding | Natural pace, narrate |
| Waiting (builds, etc.) | Speed up in post |
