# Example CLAUDE.md Templates

Real-world CLAUDE.md project templates for different tech stacks.

## Available Templates

- `CLAUDE.md` — Base template for any project
- `saas-nextjs-CLAUDE.md` — Next.js SaaS application
- `go-microservice-CLAUDE.md` — Go microservice
- `django-api-CLAUDE.md` — Django REST API
- `rust-api-CLAUDE.md` — Rust API service
- `user-CLAUDE.md` — User-level global configuration

## Usage

Copy and customize a template for your project:
```bash
cp .agent/examples/saas-nextjs-CLAUDE.md ./CLAUDE.md
# Edit to match your project specifics
```
