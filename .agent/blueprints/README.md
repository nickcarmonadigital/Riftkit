# Agent Blueprint Library

> **Pick off the shelf, build like a pro.**

This directory contains started templates ("Blueprints") for common project types. Each blueprint provides:

1. **Tailored Onboarding**: Pre-filled `project-context.md` templates.
2. **Specific Checklists**: ARA questions unique to that domain.
3. **Recommended Tech Stack**: Best-practice defaults.

## 📂 Categories

* **`01-web-and-apps`**: SaaS, Static Sites, Mobile Apps
* **`05-ai-and-ml`**: RAG Pipelines, Model Training, Agents
* **`08-plugins-and-extensions`**: VS Code Extensions, Chrome Plugins

### Planned (not yet available)

* `02-games`: Unity, Unreal, WebGL, Godot
* `03-trading-and-finance`: Bots, Algo-trading, DeFi Dashboards
* `04-web3-and-blockchain`: Smart Contracts, dApps, Tokens
* `06-hardware-and-iot`: Embedded Systems, Robotics
* `07-automation-and-devops`: CI/CD Pipelines, Infrastructure as Code
* `09-data-and-analytics`: ETL Pipelines, Dashboards

## 🚀 How to Use

1. **Run**: `Using new_project skill: Pick a blueprint`
2. **Browse**: Look inside the category folder matching your idea.
3. **Copy**: Copy the contents of the chosen blueprint into your project's `.agent/docs/0-context/` folder.
4. **Customize**: Edit the copied files with your specific details.
