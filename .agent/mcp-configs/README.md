# MCP Server Configurations

Pre-configured Model Context Protocol (MCP) server integrations for extending Claude Code capabilities.

## Usage

Copy the example config, fill in your credentials, then move to your Claude config location:
```bash
cp .agent/mcp-configs/mcp-servers.example.json .agent/mcp-configs/mcp-servers.json
# Edit mcp-servers.json — replace YOUR_*_HERE placeholders with real credentials
# mcp-servers.json is gitignored so your secrets won't be committed
```

Then customize the server entries for your environment.
