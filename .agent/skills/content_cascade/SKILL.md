# Content Cascade Skill

> **PURPOSE**: Maximize content output by extracting 30 short-form pieces from each long-form video.

## 🎯 When to Use

- After filming a long-form build/tutorial video
- Creating a content calendar from long-form
- Maximizing ROI on each piece of content

---

## 💧 The Cascade Method

One Long-Form Video → 30+ short-form pieces

```
┌──────────────────────────────────────────────────────────────┐
│                     LONG-FORM VIDEO                          │
│                  (1-6 hour tutorial/build)                   │
└──────────────────────────────────────────────────────────────┘
                            │
                            ▼
┌──────────────────────────────────────────────────────────────┐
│                    30 SHORT-FORM CLIPS                        │
├──────────────────────────────────────────────────────────────┤
│                                                               │
│  Week 1: 5-7 clips (Architecture + Setup)                    │
│  Week 2: 5-7 clips (Core Feature Builds)                     │
│  Week 3: 5-7 clips (Advanced Features)                       │
│  Week 4: 5-7 clips (Deployment + Results)                    │
│  Week 5+: 5-7 clips (Mistakes, Tips, Reflections)            │
│                                                               │
└──────────────────────────────────────────────────────────────┘
```

## 📋 Clip Categories

### Category 1: Educational (12 clips)

| Clip Type | Example |
|-----------|---------|
| Step-by-step tutorial | "How I set up the database in 2 minutes" |
| Quick tip | "The one thing I do before every build" |
| Framework/method | "My 3-step process for clean code" |
| Common mistake | "This bug cost me 2 hours—don't make it" |
| Do's and don'ts | "Do use TypeScript, don't use vanilla JS here" |
| Myth busting | "No, you don't need to memorize syntax" |

### Category 2: Visual Speed-Builds (8 clips)

| Clip Type | Example |
|-----------|---------|
| Time-lapse coding | "Watch me build the entire auth in 60 seconds" |
| Before/after | "The dashboard before vs after styling" |
| UI transformation | "Making it look premium in under a minute" |
| Feature demo | "Here's what the booking system does" |

### Category 3: Value-Forward Hooks (5 clips)

| Clip Type | Example |
|-----------|---------|
| "I built this for free" | "Agencies charge $25k for this—here's the free code" |
| Value reveal | "This one config saved me 10 hours" |
| Code giveaway | "The exact component you need—link in bio" |

### Category 4: Storytelling (3 clips)

| Clip Type | Example |
|-----------|---------|
| Build journey | "Hour 1 vs Hour 4 of this project" |
| Problem/solution | "I was stuck for 30 minutes until I tried this" |
| Reflection | "What I'd do differently next time" |

### Category 5: Results/Authority (2 clips)

| Clip Type | Example |
|-----------|---------|
| Final reveal | "Here's the finished product—100% functional" |
| Metrics/proof | "This handles 1,000 concurrent users" |

## 📍 Timestamp Extraction

While watching the long-form, mark timestamps for:

```
TIMESTAMPS.md
─────────────

## Architecture (0:00 - 0:45)
- 0:05 - Why I chose this stack (CLIP: educational)
- 0:15 - Database schema walkthrough (CLIP: tutorial)
- 0:35 - The one decision that saved me hours (CLIP: tip)

## Setup (0:45 - 1:30)
- 0:48 - Speed setup time-lapse (CLIP: visual)
- 1:10 - Environment variable trick (CLIP: tip)
- 1:25 - The mistake I almost made (CLIP: common mistake)

... continue for all sections ...
```

## ✂️ Editing Requirements

For each short-form clip:

| Element | Specification |
|---------|---------------|
| **Length** | 30-60 seconds |
| **Hook** | First 3 seconds must grab attention |
| **Captions** | Auto-generated, edited for accuracy |
| **Font** | Consistent brand font |
| **Music** | Optional, don't overpower voice |
| **End card** | CTA (follow, link in bio, comment) |

## 📅 Posting Schedule

|------|----------|------------|

| 2 | Instagram + TikTok | 5-7 |
| 3 | Instagram + TikTok | 5-7 |
| 4 | Instagram + TikTok | 5-7 |
| 5+ | Evergreen rotation | 2-3 |
