# Content Creation Skill

> **PURPOSE**: Create high-performing content following proven frameworks.

## 🎯 When to Use

- Creating long-form video content
- Writing scripts for YouTube videos
- Creating short-form content (Reels, TikTok, Shorts)
- Planning content calendar

---

## 🎬 The 7 Components of Viral Content

Every successful video has:

| Component | What It Is |
|-----------|------------|
| 1. **Topic** | The subject matter |
| 2. **Hook** | Verbal + Written + Visual (first 3 seconds) |
| 3. **Value** | The actual content/teaching |
| 4. **Format** | How it's filmed (angle changes, etc.) |
| 5. **CTA** | Call to action (follows, leads) |
| 6. **Editing** | Your consistent style |
| 7. **Structure** | Script flow pattern |

## 🪝 The Triple Hook System

Every video needs THREE hooks in the first 3 seconds:

| Hook Type | Importance | Keep Under |
|-----------|------------|------------|
| **Verbal** | 1st (most important) | 10 words |
| **Written** | 2nd | 5 words |
| **Visual** | 3rd | 3 seconds |

### Verbal Hook Formulas

```
- "I went from [BEFORE] to [AFTER] in [TIMEFRAME]"
- "The hack that got me [RESULT]"
- "[NUMBER] mistakes [AUDIENCE] make with [TOPIC]"
- "If you're struggling with [PAIN], try [SOLUTION]"
- "This is what happens when you [ACTION]"
- "Stop doing [BAD THING] if you want [RESULT]"
```

### Visual Hook Ideas

1. The clone effect
2. The sitting down
3. Multiple jumpcuts
4. Random mic on object
5. The jump in
6. The catching something
7. The performing a movement
8. Setting down phone/camera
9. The walk up
10. The mirror reflection

## 📝 Script Structures

### Comparison Structure (Best for engagement)

```
0-10 sec:  HOOK - Introduce options A vs B
10-20 sec: Option A - 3 facts (seems like best option)
20-30 sec: Option B - 3 facts (seems worse)
30-40 sec: TIME - Show passage of time
40-50 sec: RESULTS - A got bad results, B got good
50-60 sec: BREAKDOWN - Explain why
60-70 sec: CTA
```

### Step-by-Step Structure

```
0-10 sec:  HOOK - State the dream result
10-30 sec: AUTHORITY - Results you've generated, why trust you
30-40 sec: INTRO - Name the framework/method
40-50 sec: VALUE - Each step using template
50-60 sec: CTA
```

### My Story Structure

```
0-10 sec:  HOOK - Introduce you and the problem
10-20 sec: INFLECTION - Highlight the pain points
20-30 sec: RISING ACTION - Failed solutions/attempts
30-40 sec: CLIMAX - Introduce the solution
40-50 sec: FALLING ACTION - Success from solution
50-60 sec: RESOLUTION - CTA to business/service
```

## 📹 Filming Best Practices

### Pacing Rule

**More cuts = More engagement**

- Shot or cut every 2 seconds
- Types: Long shot, Medium shot, Close shot, Below shot, Above shot

### Things to NEVER Do

- ❌ Only post trending audio + B-roll (max 4/month)
- ❌ Super niche content from the start
- ❌ Only sitting and talking
- ❌ No visual interest
- ❌ Multiple niches on same page

## 📊 Content Types Mix

| Type | Description | When to Use |
|------|-------------|-------------|
| **Educational** | Value-driven, teaching | 60% of content |
| **Storytelling** | Personal stories | 15% of content |
| **Authority** | Results, social proof | 15% of content |
| **Series** | Ongoing lessons/challenge | 10% of content |

## ✅ Pre-Publish Checklist

```
SCRIPT:
[ ] Hook under 10 words (verbal)
[ ] Hook under 5 words (written)
[ ] 5th grade reading level (use Hemingway.app)
[ ] Measurable value, not fluffy
[ ] Strong CTA at end

FILMING:
[ ] Shot changes every 2 seconds
[ ] Engaging format (not just sitting)
[ ] Visual hook in first 3 seconds
[ ] Clear audio

EDITING:
[ ] Consistent fonts/colors (brand)
[ ] Captions added
[ ] Length under 60 seconds
[ ] Smooth transitions
```
