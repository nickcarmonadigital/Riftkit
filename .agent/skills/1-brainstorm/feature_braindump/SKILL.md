# Feature Braindump

**Trigger**: "Brain dump: [idea]" or "I have an idea for..."

## When to use

Use this skill when you have a raw idea and want to get straight to building or planning, without a heavy formal process.

## Process

1. **Capture**: Write down everything you are thinking about the feature.
    * What is it?
    * Who is it for?
    * How should it look?
    * Any specific tech requirements?

2. **Structure**: The AI will organize your thoughts into:
    * **Goal**: One sentence summary.
    * **User Story**: As a [user], I want [action], so that [benefit].
    * **Requirements**: Bulleted list of must-haves.
    * **Tech Notes**: Libraries, APIs, or patterns to use.

3. **Confirm**: Review the structured output.

## Checklist

- [ ] Raw idea captured
* [ ] Requirements structured
* [ ] Tech stack confirmed
