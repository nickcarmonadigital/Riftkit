# Learned Skills

This directory stores patterns and skills extracted during development sessions using the `/learn` and `/learn-eval` commands.

## Structure

```
learned/
├── README.md              # This file
├── instincts/             # Learned instincts from sessions
│   ├── personal/          # Your own observed patterns
│   └── inherited/         # Imported from teammates or repos
├── evolved/               # Skills/commands/agents generated by /evolve
│   ├── commands/
│   ├── skills/
│   └── agents/
└── *.md                   # Individual learned pattern files
```

## How to Use

- `/learn` — Extract a reusable pattern from the current session
- `/learn-eval` — Extract with quality evaluation before saving
- `/instinct-status` — View all learned instincts
- `/instinct-import` — Import instincts from teammates
- `/instinct-export` — Export instincts for sharing
- `/evolve` — Cluster related instincts into skills/commands/agents
