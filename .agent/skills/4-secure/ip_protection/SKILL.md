## The Toyota Doctrine

> **Principle**: Patent the IMPLEMENTATION, not the CONCEPT.

```
PUBLIC CONCEPTS (Can't Patent)          TECHNICAL SYSTEMS (Can Patent)
──────────────────────────────          ─────────────────────────────────

Hormozi's 101/201 framework      →→→    AI Context Extraction Engine
Hormozi's scaling thresholds     →→→    Automated Threshold Detection System
Caleb's branding pillars         →→→    Content Cascade AI Generator
BPO knowledge management         →→→    8-Brain Modular Architecture
Industry SOPs                    →→→    SSoT Living Document System

THE PATENT IS ON THE RIGHT SIDE, NOT THE LEFT
```

## Before Any GitHub Release Checklist

```
GITHUB RELEASE IP REVIEW
────────────────────────

BEFORE PUSHING ANY CODE:

[ ] Review: Does this code reveal trade secrets?
    - 91 questions text
    - Prompt engineering
    - Threshold values
    - Vertical customizations

[ ] Review: Is there patentable innovation here?
    - If yes, file provisional BEFORE pushing
    - You have 1 year in US after public disclosure
    - In most other countries: disclosure = no patent

[ ] Sanitize: Remove any sensitive content
    - API keys (obviously)
    - Internal business logic
    - Customer data schemas
    - Proprietary algorithms

[ ] Add: Proper licensing
    - README with license terms
    - LICENSE file
    - Terms of use if applicable
```

```
[ ] Patent consideration
    - If showing novel system, file provisional first
    - Document date of first public disclosure

[ ] Competitive review
    - Would a competitor learn something valuable?
    - Could they replicate your advantage?
```

## Quick Decision Matrix

```
SHOULD I PATENT THIS?
─────────────────────

Is it a technical implementation?
├── NO → Can't patent (it's a concept)
└── YES → Continue

Is it novel (no prior art)?
├── NO → Can't patent
└── YES → Continue

Would disclosure help competitors?
├── YES → Consider TRADE SECRET instead
└── NO → Continue

Is it worth $10,000-15,000 to protect?
├── NO → Trade secret or copyright only
└── YES → FILE PROVISIONAL NOW
```
