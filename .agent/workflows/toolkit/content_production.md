# Content Production Workflow

**Purpose**: High-volume content machine workflow (Waterfall Method).
**Goal**: 1 Long-form video -> 30+ Short-form assets.

---

# Phase 1: Ideation & Research

**Goal**: Validated content ideas.

1. **Signal Check**
   - Check comments/DMs for questions.
   - Check trending topics in niche.
   - Review `video_research` skill output.

2. **Concept Selection**
   - Select ONE core topic (The Magnum Opus).
   - *Criteria*: Must be valuable enough for 10+ minutes.

---

# Phase 2: Blueprinting (Script/Outline)

**Goal**: Structure for filming.

1. **Hook Construction**
   - Write 3 variations of the intro.
   - "In this video, I'm going to show you..."

2. **Body Outline**
   - Beat 1: The Problem.
   - Beat 2: The Old Way.
   - Beat 3: The New Way (The Build).
   - Beat 4: The Results.

3. **Asset Prep**
   - Prepare code examples.
   - Prepare diagrams (Excalidraw).

---

# Phase 3: Production (Filming)

**Goal**: High-quality raw footage.

1. **A-Roll (Talking Head)**
   - Film intro/outro + key concepts.
   - Look at lens.

2. **B-Roll (Screen Recording)**
   - Record the build/screen.
   - Zoom in 125% for readability.
   - Hide bookmarks/personal info.

---

# Phase 4: The Build (Waterfall Source)

**Goal**: The "Meat" of the content.

*If this is a coding video:*

1. **Code the solution live/walkthrough.**
2. **Narrate thought process.**
3. **Capture "Mistakes"** - these make great shorts.

---

# Phase 5: Waterfall Extraction

**Goal**: 30 Pieces of Content.
**Skill**: Use `content_waterfall`.

1. **Timestamping**
   - Watch raw footage.
   - Mark in/out points for shorts.

2. **Categorization**
   - Educational (Tutorials)
   - Entertainment (Stories/Mistakes)
   - Visual (Speed builds)

3. **Editing**
   - Edit Long-form first.
   - Edit Shorts batch (Vertical 9:16).

---

# Phase 6: Distribution

**Goal**: Maximum Reach.

1. **Primary Launch**
   - YouTube Long-form.
   - Newsletter blast.

2. **The Drip (Weeks 1-4)**
   - Schedule daily shorts (TikTok/Reels/Shorts).
   - Post LinkedIn/Twitter text threads adapted from script.

3. **Community**
   - Reply to comments (First 1 hour critical).

---
