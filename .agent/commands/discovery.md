# Discovery Command

Alias for the `/atom` command, focused on **systematic blind spot discovery**.

See `/atom` (`.agent/commands/atom.md`) for full documentation.

## Quick Reference

- **Skill**: `.agent/skills/toolkit/age/SKILL.md`
- **Purpose**: Systematic discovery of unknown unknowns and blind spots
- **Aliases**: `/atom`, `/age`, `/gap-analysis`, `/adversarial-audit`, `/discovery`

## When to Use

Use `/discovery` when you want to:
- Find unknown unknowns in your project
- Discover blind spots in your thinking or implementation

- Run a fresh-eyes analysis of an existing system
