# Gap Analysis Command

Alias for the `/atom` command, focused on **capability gap discovery**.

See `/atom` (`.agent/commands/atom.md`) for full documentation.

## Quick Reference

- **Skill**: `.agent/skills/toolkit/age/SKILL.md`
- **Purpose**: Identify what's missing, incomplete, or underspecified
- **Aliases**: `/atom`, `/age`, `/gap-analysis`, `/adversarial-audit`, `/discovery`

## When to Use

Use `/gap-analysis` when you want to:
- Find missing capabilities or features
- Identify incomplete implementations
- Discover what's underspecified in requirements or architecture
- Validate coverage of testing, security, or documentation
