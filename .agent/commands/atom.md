# ATOM Command

This command invokes the **ATOM (Axiomatic Thinking for Omnidirectional Meta-analysis)** protocol for systematic gap discovery and analysis.

## What This Command Does

1. **Systematic Gap Analysis** - Runs up to 53 independent discovery loops against a target (codebase, architecture, process, or system)
2. **Multi-Method Coverage** - Applies 51 analysis methods drawn from AI/ML research, classical engineering, and adversarial thinking
3. **Zero-Context Loops** - Each loop starts fresh (the "Ralph Wiggum Protocol") to avoid anchoring bias
4. **Gap Log** - Produces a prioritized gap log with findings, severity, and recommended actions

## When to Use

Use `/atom` when:
- You need a comprehensive audit of what's missing or broken
- Starting a new phase and want to validate completeness
- Performing adversarial analysis or red-teaming
- Looking for blind spots in architecture, security, testing, or process
- Running a capability gap analysis

## How It Works

The ATOM protocol will:

1. **Select target scope** - What system/codebase/process to analyze
2. **Run discovery loops** - Each loop applies a different analytical method
3. **Accumulate findings** - Build a gap log across all loops
4. **Deduplicate and prioritize** - Merge overlapping findings and rank by severity
5. **Produce recommendations** - Actionable next steps for each gap
