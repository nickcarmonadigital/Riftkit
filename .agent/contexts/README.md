# Dynamic Contexts

System prompt contexts for different development modes. Switch context to optimize Claude's behavior for specific tasks.

## Available Contexts

- **dev.md** — Development mode: building features, writing code
- **review.md** — Review mode: code review, quality analysis
- **research.md** — Research mode: exploration, investigation

## Usage

Reference a context to activate it:
```
Use the dev context from .agent/contexts/dev.md
```
